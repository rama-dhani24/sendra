Sendra
Sendra is a cross-currency payment system that allows users to send money in different currencies (GBP, USD, EUR, USDT) and convert it to local currency (TZS) on the receiving side.

Project Overview
The system handles currency conversion using real-time FX rates and uses USDT as a settlement layer for transfers.

How it works
User sends money in GBP, USD, EUR, or USDT
If fiat currency is used, it is converted to USDT using FX rates
USDT is then converted to TZS for the receiver
If USDT is sent directly, no conversion is needed
